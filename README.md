📈 Bitcoin Market Direction Prediction using Sentiment Analysis

A machine learning project that predicts the next-day Bitcoin market direction using Twitter sentiment analysis and market features.

This project combines Natural Language Processing (NLP) with financial time-series data to analyze whether public sentiment about Bitcoin can help predict future market movement.

The model is deployed using Streamlit, allowing users to input tweets and receive a prediction of the next day's BTC market direction.

📈 Bitcoin Market Direction Prediction using Sentiment Analysis
📌 Project Overview

This project predicts the next-day direction of Bitcoin price (UP or DOWN) using Twitter sentiment analysis and market features.

The model analyzes:

Public sentiment from Bitcoin-related tweets

Historical Bitcoin price data

Market volatility and trading volume

These features are used to train a machine learning model that predicts whether the Bitcoin market will go up or down tomorrow.
